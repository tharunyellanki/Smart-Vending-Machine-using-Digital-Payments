# Smart Vending Machine Using Digital Payments

## Overview
This project is a **Smart Vending Machine** designed to automate product dispensing through **digital payment verification**. The system uses **ESP32** to connect to the internet and verify successful payments through an **IMAP mail server**. Once payment confirmation is received, the selected product is automatically dispensed.

This project demonstrates the integration of **Embedded Systems, IoT, Cloud Communication, and Automation**.

---

## Features
- Cashless product purchasing system
- Digital payment verification through IMAP server
- Automatic product dispensing after successful payment
- Wi-Fi enabled using ESP32
- Fast and secure transaction process
- Low-cost smart automation solution

## Working Principle
1. User selects the product.
2. QR code / payment request is generated.
3. User completes digital payment.
4. ESP32 connects to IMAP server through Wi-Fi.
5. Payment confirmation email is verified.
6. Product dispensing motor is activated.
7. Product is delivered to the user.
